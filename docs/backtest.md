# 回測與績效

本文件說明本專案如何評估策略，以及目前 `00631L + 00830` 的回測結果。數字應以重新執行 `go run ./cmd/eval_csv` 的輸出為準。

## 評估情境

回測採用「期初資金 + 每月解鎖新資金」：

- 期初資金：`initial_cash`，預設 100,000。
- 每月注資：`monthly_contribution`，預設 2,500。
- 注資日：每個日曆月第一個交易日。
- 起始月不額外注資。
- 交易成本目前未計入。

這個設定用來模擬長期定期投入資金的使用情境，而不是一次性滿倉。

## 指標定義

| 指標 | 做法 | 說明 |
| --- | --- | --- |
| MWR / XIRR | 把期初投入與每月注資視為流出，期末清算視為流入，解年化內部報酬率 | 反映每筆錢實際投入多久 |
| NAV 最大回撤 | 注資視為用當日淨值申購新單位，再計算單位淨值最大回撤 | 排除持續注資對權益曲線的灌水 |
| Calmar | `MWR / abs(MaxDD)` | 衡量風險調整後表現 |
| 資金利用率 | 平均持股市值佔總權益比例 | 檢查低回撤是否只是大量抱現金 |

## 對照組

回測同時比較三種組合：

- 策略：使用 `trading.Engine` 執行目前買賣規則。
- B&H：資金一解鎖就等權買滿追蹤標的，持有到期末。
- Blend：與策略平均曝險相同的市場與現金定權重組合，用來檢查策略是否有真實擇時能力。

三者使用完全相同的起始資金、注資時程、視窗與交易日歷。

## 重現指令

先下載 TWSE CSV 快取：

```bash
go run ./cmd/fetch_data
```

再執行離線評估：

```bash
go run ./cmd/eval_csv
```

若要使用 DB 資料：

```bash
go run ./cmd/evaluate
go run ./cmd/research_run
```

## 目前結果

追蹤標的為 `00631L + 00830`，區間約為 2019-05 到 2026-06。`00631L` 是 2 倍槓桿 ETF，現行參數已針對其波動特性重新調校。

決策基準為**開盤價**（`decision_price_basis: open`）：以當日開盤價成交、指標只看到前一交易日收盤（見 [strategy.md](strategy.md)）。回測與線上同此基準。下列數字為開盤基準、為其重新調參後（`regime_ma_window 85`、`trail_stop_bear 0.08`、`00631L` 覆寫 60）的結果。

### 全期連續回測

投入本金合計為 312,500。

| 指標 | B&H | 策略 |
| --- | --- | --- |
| 期末總權益 | **2,798,385** | 1,972,943 |
| 期末閒置現金 | 約 0 | 335,359 |
| 本金倍數 | **8.95x** | 6.31x |
| MWR | **+48.8%** | +40.5% |
| NAV 最大回撤 | -51.4% | **-30.2%** |
| Calmar | 0.95 | **1.34** |
| 資金利用率 | 100.0% | 68.2% |
| 買/賣次數（停利/了結） | 86/0 | 346/45（39/6） |

策略的報酬低於槓桿 B&H，但回撤明顯較小，Calmar 較高。

### Walk-forward 穩健性

設定為 24 個月視窗、每 3 個月步進，共 21 個視窗。

| 指標 | 策略 | B&H | Blend |
| --- | --- | --- | --- |
| 中位 MWR | +32.5% | **+38.2%** | +27.6% |
| 中位 MaxDD | **-21.5%** | -43.3% | - |
| 中位 Calmar | **1.66** | 1.08 | - |
| 資金利用率 | 68.7% | 100.0% | - |
| Calmar 勝率 vs B&H | **100.0%** | - | - |
| 真擇時勝率 vs Blend | **90.5%** | - | - |

目前 scorecard 四道核心關卡（報酬參與、回撤降低、Calmar 勝率、真擇時）皆通過。

### IS/OOS 樣本內外驗證

`cmd/eval_csv` 會執行滾動式 OOS 驗證，用來檢查策略是否只在樣本內好看。現行（開盤基準）配置 OOS 中位 Calmar 約 1.91，約為 IS Calmar 2.05 的 **93%**，三折 OOS 皆通過 5/5 或 4/5 關卡，判讀為「穩健」，表示樣本外未退化。

> 註：直接把收盤基準的參數套到開盤基準會明顯退化（Calmar 1.51→1.12、walk-forward 由通過轉為不通過、OOS 保留率掉到 ~48%）。因此開盤基準是另外以 walk-forward / IS-OOS 重新調參（`regime_ma_window 95→85`、`trail_stop_bear 0.10→0.08`），落在「reg 80~85 × trail 0.08」一整片皆通過的平台上，而非單點最佳。

per-stock 覆寫採用準則：

- 只接受 IS 表現良好且 OOS 不退化的覆寫。
- `00631L` 使用 `regime_ma_window: 60`。
- `00830` 維持共用 `regime_ma_window: 85`，因為更長 regime 在 OOS 退化。

## 注意事項

以上結果未計交易成本，且追蹤標的為事後挑選且長期向上的 ETF。結果適合作為工程與研究參考，不代表未來績效，也不構成投資建議。
