# 各方法逐一掃描的實驗結果（完整數據）

> 每個旋鈕都在 **walk-forward（24 月視窗 / 步進 3 月 / 21 個視窗，006208+00830）** 下、
> 套在**原始 baseline** 上做參數掃描。下表為各候選值的中位數指標。
> 判定規則：勝者必須**顯著提升風險調整報酬**，且通過護欄（① 中位 CAGR 不比 champion 差 >12% ② 真擇時不比 champion 低 >8pp）。

**Baseline 參考列**：CAGR **+10.1%** ｜ MaxDD **−12.9%** ｜ Calmar **1.01** ｜ 曝險 **35.5%** ｜ 真擇時 **61.9%**

關鍵讀法：很多旋鈕的 **Calmar 看起來更高**，但那是「**少買、抱現金**」換來的——CAGR（報酬）同時崩掉，
所以被護欄擋下。真正的勝者要「Calmar 變好**且**報酬不掉」。

## 總結

| 類別 | 旋鈕 | 結果 |
|---|---|---|
| **免費午餐** | sell_rsi_min=75 | ✅ Calmar↑、報酬不變 → **採納** |
| 風險換報酬 | ma_window↓ / multiplier↑ | △ 報酬↑但回撤↑ → 保留為選項 |
| 濾網類 | long_ma / buy_rsi / bias / confirm_up | ✗ 砍掉深跌 alpha → 丟棄 |
| 刻度盤類 | cooldown / sell_threshold / cash_floor | ✗ 無甜蜜點 → 丟棄 |

**一句話**：這個策略的 edge 是「無差別買最深的回檔」，任何「進場端加謹慎」都在砍 edge；
唯一能無痛改善的是「**出場端賣在強勢**」。要更高報酬只能接受更大回撤（composite 路線）。
