# 多頭部署:少次大額 vs 小額高頻 (計入交易成本)

- 膝點核心 + 多頭固定大額 (bull_buy_amount,已×mult=3) × 多頭冷卻 (bull_cd)
- 成本:手續費 0.1425%/邊 + 證交稅 0.10%/賣;『交易/窗』=每視窗買賣次數中位數 (換手指標)
