# 目前最佳交易方法 — 詳細報告

> 牛熊雙策略 + 熊市移動停利的逢低加碼法。本報告可直接復現。
> 狀態:**回測驗證的最佳方案**(冷卻=2 周約束下);尚未全部寫入 config.yaml,正式上線前建議先做 IS/OOS。

## 一、摘要

一套針對台股大盤型 ETF (006208、00830) 的長線策略。核心思路:**看股市天氣換打法**——
多頭(上升趨勢)時主動用固定大額把資金部署出去;空頭(下降趨勢)時分級抄底,但用 8% 移動停利保住獲利、壓低回撤。

**績效(walk-forward 21 視窗;未計交易成本):**

| 指標 | 現況(原始保守版) | **本方法** | Buy & Hold |
| --- | --- | --- | --- |
| 年化報酬 CAGR | +11.9% | **+26.1%** | +24.7% |
| 最大回撤 MaxDD | −15.3% | −22.7% | −32.8% |
| Calmar(報酬/回撤) | 1.10 | **1.25** | 0.94 |
| 資金利用率(平均曝險) | 40% | **81%** | 100% |
| 交易/視窗 | 少 | 66 | 1 |

→ **報酬翻倍、追平 Buy & Hold,但最大回撤只有 B&H 的 ⅔,且風險調整(Calmar)更佳。**

## 二、核心理念

| | 多頭🟢(價>200日均線) | 空頭🔴(價≤200日均線) |
| --- | --- | --- |
| 信念 | 未來上漲機率大 → 盡量把錢部署出去 | 最賺的買點在下跌段 → 分級抄底 |
| 買進 | 寬鬆、固定大額 | 嚴格、跌越深買越多 |
| 賣出保護 | 無(讓贏家奔跑) | 8% 移動停利(賺過 20% 後啟動) |

## 三、完整交易規則(可直接復現)

**通則**:每檔股票各自獨立;每個交易日用**當天收盤價**判斷;只能用手上現金、**不可借錢**(算出的股數買不起就買買得起的部分)。

需要的指標:**10 日均線、200 日均線、近 252 日最高收盤、RSI(14) Wilder 法、持有期間最高收盤**。

### 步驟 A — 判斷牛熊
算 200 日均線。今天收盤 **> 200MA → 牛市🟢**;否則 **→ 熊市🔴**(資料不足 200 天當熊市)。

### 步驟 B — 要不要買
**買進需同時滿足:**
1. 有有效收盤價、且已有 10 日均線。
2. 價格門檻:🟢牛市 收盤 **< 10MA × 1.05**;🔴熊市 收盤 **< 10MA**。
3. 冷卻:距上次買這檔 **≥ 14 天**(牛熊都 14)。

**買多少:**
- 🟢**牛市:固定 6,000 元**(不分跌深跌淺)。
- 🔴**熊市:看「離近 252 日最高點跌幾 %」分級:**

  | 離一年高點跌幅 | 買進金額 |
  | --- | --- |
  | 0–10% | 1,500 |
  | 10–20% | 3,000 |
  | 20–30% | 6,000 |
  | 30–40% | 12,000 |
  | 超過 40% | 24,000 |

- 換算股數 = 四捨五入(金額 ÷ 收盤價),再用「買得起」上限(含手續費)夾一次。

### 步驟 C — 要不要賣(買完後、用更新後持股再判斷)
依序檢查,先中先賣:

**規則1 — 熊市移動停利(只在🔴空頭):**
- 啟動條件:這檔**曾經賺過 ≥ 20%**(持有期間最高收盤 ÷ 持倉最低成本 − 1 ≥ 0.20)。
- 觸發:今天收盤 **≤ 持有期間最高收盤 × (1 − 8%)** → **把這檔全部賣光**。
- 🟢牛市不設此條。

**規則2 — 獲利了結 +100%(牛熊都有):**
- 觸發:**持倉最低成本獲利 ≥ 100%** 且 **RSI(14) ≥ 75**(過熱才賣)。
- 賣出量:**30,000 元**對應股數,從成本最低批次開始賣。

### 步驟 D — 更新狀態
有持倉就更新「持有期間最高收盤」;全部賣光則歸零、冷卻重算。

  - {above: -0.1}
  - {above: -0.2}
  - {above: -0.3}
  - {above: -0.4}

# 賣出:獲利了結 + 熊市移動停利

## 五、這套是怎麼演化來的(每個改動的貢獻)

| 改動 | 解決什麼 | 效果 |
| --- | --- | --- |
| 原始 Baseline | — | 只逢低買、+100% 才賣 → 太保守、只用 40% 資金、報酬輸大盤 |
| **賣在 RSI 強勢** (sell_rsi=75) | 賣太早 | Calmar 1.01→1.08(免費) |
| **加碼曲線加陡** (ratio=2) | 加碼太淺 | +報酬,小幅 +回撤 |
| **深度基準改 peak** | 「相對自己成本」太粗糙 | 依市場折價加碼,利用率↑、擇時↑ |
| **牛熊 regime + 牛市固定大額** | 多頭抱現金、利用率低 | 利用率 40%→80%、報酬翻倍 |
| **熊市 8% 移動停利** | 一路抱著住套牢、回撤大 | 回撤主控,把高曝險的回撤壓回 −22% |

## 六、風險與限制(務必正視)

1. **選擇性過擬合**:本配置是從數百組掃描中選出的最佳,天生樂觀。
2. **倖存者/期間偏誤**:006208、00830 是事後挑選、長期上漲、近年多頭的標的。換到走勢差或長空標的,
   逢低加碼與移動停利的表現會明顯不同。
3. **尚未做樣本外驗證(IS/OOS)**:上線前應「前段挑參數、後段驗證」確認不是只在歷史上好看。
4. **未計交易成本**:回測未計手續費、證交稅、滑價、零股限制、配息再投入(經測試手續費影響極小,約 0.2 個百分點,故已移除成本機制以簡化)。
5. **冷卻=14 的取捨**:2 周冷卻買得較勤、曝險與回撤偏高;放寬到 3 周(冷卻=21)其實 Calmar 更高(1.40)、
   回撤更低(−20.8%)、交易更少(43)。本報告是「冷卻=2 周」約束下的最佳。
